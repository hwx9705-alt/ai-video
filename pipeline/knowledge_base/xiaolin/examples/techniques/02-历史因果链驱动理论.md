# 技巧 2：用历史因果链驱动理论讲解

## 什么时候用
要讲一个抽象理论/概念/制度时。不要"X 理论是什么"，要"什么历史困境逼出了 X 理论"。

## 真实片段（《国债》· 经济学思想演进）

> "现代经济学鼻祖亚当·斯密非常反对政府花钱，觉得市场竞争才是效率最高的。美国 18、19 世纪的时候非常尊重他这套理论，政府基本能不管就不管。
>
> 但慢慢问题就出现了——完全让市场自由竞争，**每隔一段时间经济就崩一次**。最严重的就是 1929 年那个**大萧条**，持续了十多年。事后很多经济学家都认为，这个萧条之所以持续这么长，和政府的不作为脱不了干系。
>
> 就是在这种时代背景之下，出现了另一个经济学大神——**凯恩斯**。他就提出来一套新理论：市场也会失灵，政府你不能完全不管，你得管，你得刺激它。"

## 技法拆解
1. 理论**嵌入历史故事**，观众不知不觉学会
2. 每个理论出场前，先铺"**上一套理论为什么不够用**"
3. 用时间/事件推进：斯密 → 大萧条 → 凯恩斯 → MMT
4. **理论就是故事的一部分**，不是课本条目

## 其他案例
- 《美联储》：美国 250 年一半时间没央行 → "谁都能印钱" → "JP 摩根以个人之力救金融危机" → 所以终于建立美联储
- 《关税》：李嘉图理论 → 第一次全球化（英国开放）→ 斯姆特-霍利关税 → 贸易寒冬 → GATT/WTO

## 禁忌
- ❌ "凯恩斯主义认为政府应该干预经济，用财政政策刺激总需求…"（教科书式）
- ✅ "大萧条持续了十多年，经济学家才意识到 —— 政府不能再'啥也不管'了，得有人来管，凯恩斯就这么被逼出来的"
