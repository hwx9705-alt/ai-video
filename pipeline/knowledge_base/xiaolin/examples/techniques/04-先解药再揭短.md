# 技巧 4：先给解药再揭短

## 什么时候用
讲一个限制/问题时，先让观众放松（看到解决方案），再深挖揭示这个方案的边界。制造"层层剥开真相"的感觉。

## 真实片段（《国债》· 利率红线 → 通胀红线）

> "发太多 → 利率升 → 成本升 → 债务漩涡。听起来好像政府就完蛋了？别急，**政府有一招可以对抗利率上升**。
>
> 日本不是从 2013 年开始搞安倍经济学嘛，央行负责印钱买国债——需求增加了，价格抬高，利率就被压下去了。直到 2020 年，日本政府压根装都懒得装了：我就搞无限量 QE，目标就摆明面上，**10 年期利率压在 0.25 以下**。
>
> **可是天下哪有免费的午餐**。有一个限制是政府永远不可能绕过去的，这也是政府、央行、经济学家谈之色变的一件事 —— **就是通货膨胀**。"

## 技法拆解
1. **先给解药**（央行配合 QE 可以绕过利率）让观众放松
2. **再揭短**（天下没有免费午餐 → 通胀才是终极限制）
3. 过渡句："可是天下哪有免费的午餐" 是**经典转折句式**
4. 观众始终"有新的发现"，层层剥开真相

## 其他案例
- 《关税》："保护幼稚产业是很好的理由" → "但一旦这个产业成熟了还拆不掉关税，就会变成既得利益"
- 《美联储》："QE 可以印钱救市" → "但出现了药物依赖（Taper Tantrum）" → "最后通胀怪兽觉醒"

## 禁忌
- ❌ "政府想绕过利率限制需要印钱，但这样会导致通胀"（一句话把两层都塞进去，失去张力）
- ✅ 先铺解药（QE 能压利率）→ 写得让观众以为"那就解决了嘛" → 然后一句"可是天下没有免费的午餐" 转折
