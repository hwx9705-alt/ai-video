  一、项目现状（接手时）

  骨架代码完成，dry-run 可跑通。所有 Agent 的 LLM 调用是占位实现（STUB），没有实际产出。具体问题：

  - OpenAI 库依赖安装失败（Anaconda SSL 模块损坏）
  - API 密钥未配置
  - Python 3.8 环境有 type hints 兼容性问题
  - Script Agent 的 prompt 是抽象描述，无 few-shot examples
  - Storyboard Agent 产出无法解析为结构化 JSON
  - 无 Web UI，只有 CLI

  2.3 Script Agent 优化（核心）

  在 OUTLINE_SYSTEM_PROMPT 和 SCRIPT_SYSTEM_PROMPT 中嵌入了小Lin说的真实脚本片段作为 few-shot examples：

  嵌入的三类示例：

  1. 开头 Hook（来自《一口气了解美联储》）
  ▎ "美联储可以说是这个星球上对金融市场影响力最大的机构。政策出点问题就导致了20世纪最严重的全球经济大萧条……这12个人开开
  会动动嘴皮子决策就能影响全球经济。而就是这么重要的机构，它竟然还有股东，每年还会分红。"
  2. 数据两步翻译法（来自《一口气了解国债》）
  ▎ "国债比GDP 206%，冠绝全球。这个数字是什么概念？就是日本所有人赚的钱，2.6年才够还他政府的债。你要是按日本政府2022年财
  政收入算，那他就是不吃不喝，完全不花钱，一分钱也不花，18年才能还清这些债。"
  3. 段落转折黄金句式（来自《一口气了解国债》）
  ▎ "好那你说，就算政府它可以借钱花钱，难道说你就这么无止境的借钱这么花下去吗？对吧，这听着肯定不合理……因为它具备一个你
  我都不具备的能力——就是印钱。"

  同时将 10 条技巧库从抽象描述改为带实际脚本例句的具体说明。

  2.4 Storyboard Agent 解析层

  新增 _parse_storyboard_output 方法，支持三种 LLM 输出格式：
  - 两个独立 JSON 代码块（style_seeds + storyboard 数组）
  - 一个大 JSON 对象包含 style_seeds 和 storyboard 两个 key
  - style_seeds 直接在顶层（有 color_palette 字段）

  实测解析成功：7个分镜段，style_seeds 含 4 个字段，全部正确提取并存入 State。
